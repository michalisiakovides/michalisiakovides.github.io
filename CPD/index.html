About Me.

PERSONAL PROJECTS

Lidar Simulation for Simultaneous Localisation and Mapping (SLAM)

Lidar simulation software, enabling precise robotic mapping and navigation through sensor modelling, dynamic visualisation, and obstacle detection for Simultaneous Localisation and Mapping (SLAM) applications.

Full-Body Motion Capture with 3D model with a Web Application

ESP32-based full-body motion capture system using IMUs, that transmits live data to a web application, enabling real-time visualisation of a 3D model and tracking of human positions and postures.

UNIVERSITY PROJECTS

Self-Driving Car Navigation in a Mini Town with a Web Application

[Report]

Computer Vision-based self-driving car with the ability to navigate in a simulated and real-life mini-town, while showcasing real-time data visualisation through a user-friendly web application. The project was part of the 3rd year individual project.

Embedded Systems Project

STM32-based line following buggy capable of navigating through various tracks. My role in the team was to design the PCB to integrate IR sensors for the line-following functionality.

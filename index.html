Leo Rover Group Project

[View 3D Model]

My role in the team was to develop the navigation and mapping algorithms using ROS2 Python. The robot was running an EKF by utilising its LiDAR,wheel encoders, and IMU to generate a map of its surroundings and localise itself. A Star Algorithm

The A* path planning algorithm was implemented within a graphical user interface using Python. The user can place start and end points, as well as obstacles, in the GUI and visualise the shortest path.

Handwritten Digit Recognition using CNN

A Convolutional Neural Network(CNN) was built to classify handwritten digits using the MNIST dataset. The CNN was based on the classic LeNet-5 architecture. PyTorch was used to implement and evaluate the CNN, achieving over 98% accuracy across training, validation, and testing phases.

COMPETITIONS

UoM Robotics Society Hack-A-Bot2024

A basic swarm consisting of 2 robots was deployed to navigate and map the entire maze using the MONA robots. My role in the team was to develop localisation software. This software utilised infrared proximity sensors to determine the position of the walls relative to a global frame and the robot's orientation. Additionally, I worked on communication software utilising NRF24 radio modules for ground control and communication between the agents.

UoM Robotics Society Hack-A-Bot2023

A basic swarm consisting of 2 robots was developed to play football against another team using the MONA Robots. My role in the team was to develop communication software (NRF24 radio modules) for ground control and the agents.

American Astronautical Society (AAS) CanSat

Can-sized satellite equipped with microcontrollers, sensors, and communication equipment to perform various tasks, such as collecting data and transmitting it to a ground station. My role in the team was to select the necessary sensors and design the PCBs.
